# 📊 Sales Performance Dashboard

## Overview
This project analyzes sales data to identify revenue trends, profit margins, and top-performing products.

## Features
- Monthly Sales Analysis
- Revenue Analysis
- Profit Analysis
- Top Products
- Interactive Dashboard

## Tools Used
- Google AI Studio
- HTML
- CSS
- JavaScript
